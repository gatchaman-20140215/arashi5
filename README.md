# arashi5
太平洋の嵐5のセーブデータを書き換えるためのプログラム。
http://www.ss-alpha.co.jp/products/arashi5.html
セーブファイルをExcelに書き出す機能とExcelをセーブファイルに書き戻す機能があります。
